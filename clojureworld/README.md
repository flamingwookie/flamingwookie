# clojureworld

A complete world created in Clojure - with a REST interface!

## Usage

(use 'clojureworld.rest)

(start)

[http://localhost:5000/](http://localhost:5000/)
or
[http://localhost:5000/hello-world](http://localhost:5000/hello-world)
